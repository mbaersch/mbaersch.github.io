Testseite für walker.js (Demo mit httpbin-Requests)

Hier wird eine lokale Fassung von walker.js geladen und im Code als Destination httpbin.org/anything und die Console definiert. Der Endpunkt bekommt das komplette Event als JSON Payload gesendet, wie es auch im elbLayer zu finden ist. Empfangene Daten werden von diesem Demo-Endpunkt wieder zurück gesendet und können in im Netzwerk-Tab kontrolliert werden.

Weitere Hinweise zur Verwendung von walker.js siehe https://docs.elbwalker.com/.
